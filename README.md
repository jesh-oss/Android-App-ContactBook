# 📱 Contact Book App

## 🚀 Overview

The Contact Book App is a simple Android application that allows users to store and manage contacts efficiently. Users can add new contacts, view all saved contacts, and quickly access the most recent entry.

---

## ✨ Features

* ➕ Add new contacts (Name & Phone Number)
* 📋 View all saved contacts
* 🕒 View the most recently added contact
* 💾 Local database storage using SQLite
* 🎨 Clean and modern UI using Material Design

---

## 🛠 Tech Stack

* Java
* XML
* Android Studio
* SQLite Database
* Material Design Components
